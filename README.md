# 🤖 RUTINATOR - AI Fitness & Nutrition Bot
> **Versión 1.0 (MVP) - Funcional**

RUTINATOR es un asistente inteligente de Telegram diseñado para automatizar la creación de planes de entrenamiento y nutrición personalizados.

## 🚀 Funcionalidades Actuales
- **Onboarding Conversacional**: Flujo dinámico con FSM.
- **Progresión Semanal Automática**: Lógica de carga y descarga.
- **Generación de PDF en Memoria**: Reportes rápidos y limpios.
- **Revista Nutricional**: Módulo "Come Como Entrenas" integrado.

## 📦 Instalación
1. `pip install -r requirements.txt`
2. Configura tu `.env`
3. `python src/bot_telegram.py`
